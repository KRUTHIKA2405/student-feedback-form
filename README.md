# Student Feedback Registration Form
A full-stack feedback system with automated testing and CI/CD integration.

## Features
- **HTML5/CSS3**: Responsive and user-friendly UI.
- **JavaScript**: Client-side validation for email, mobile, and word count.
- **Selenium**: Automated test suite for form functionality.
- **Jenkins**: Configured for continuous integration.

## How to Run
1. Open `index.html` in any browser to view the form.
2. To run tests:
   - Install dependencies: `pip install -r requirements.txt`
   - Run: `python test_form.py`
